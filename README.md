## Templates & Starters

Open-source templates ready to clone and customize.

- [nuxt-ui-templates](https://github.com/nuxt-ui-templates/portfolio) - Minimalist portfolio template crafted with **Nuxt UI** and **Nuxt Content**.
- [portfolio-template](https://github.com/AkberJag/portfolio-template) - Fully customizable template built using Vue.js and **Tailwind CSS**.
- [portfolio-vuejs](https://github.com/hbollon/portfolio-vuejs) - Connected portfolio using Vue, **CosmicJS** (Headless CMS), and Bootstrap.
- [vue-developer-portfolio](https://github.com/Asfak00/vue-developer-portfolio) - Modern UI/UX template with dynamic content management and SEO optimization.
- [vue-resume-template](https://github.com/ryanbalieiro/vue-resume-template) - A clean, resume-focused template built with **Vue 3 Composition API**.
- [vuefolio](https://github.com/ralodev/vuefolio) - Vue.js + **i18n** portfolio template, easy setup and SEO friendly.

## Personal Portfolios

Real-world examples of developer portfolios for inspiration.

- [claudiabdm](https://github.com/claudiabdm/portfolio-2021) - Built with **Nuxt 3** and **Storyblok CMS**.
- [developer-portfolio](https://github.com/xtenzQ/developer-portfolio) - Clean, flat-style developer portfolio with **Bootstrap 5**.
- [eggsy](https://github.com/eggsy/website) - Full-featured website written in **Nuxt**, TypeScript, and **Windi CSS**.
- [Eldin-Space-Vue](https://github.com/EldinZaimovic/Eldin-Space-Vue) - Personal web portfolio by Eldin Zaimovic.
- [Hetari](https://github.com/Hetari/portfolio) - UI/UX clone implementation focused on pixel-perfect design.
- [jeferson-sb](https://github.com/jeferson-sb/portfolio) - Personal website leveraging **Vue 3** and **Vite**.
- [mahy209](https://github.com/mahy209/vuejs-portofolio) - Simple personal portfolio implementation.
- [mayashavin](https://github.com/mayashavin/portfolio-new) - Personal website of Maya Shavin, showcasing advanced Vue patterns.
- [MuhaddiMu](https://github.com/MuhaddiMu/Portfolio) - **Nuxt-based** modern portfolio website.
- [pablosirera.com](https://github.com/pablosirera/pablosirera.com) - Personal portfolio of Pablo Sirera.
- [realstoman](https://github.com/realstoman/vuejs-tailwindcss-portfolio) - Multipage responsive theme with **Dark Mode** support using Tailwind.
- [suradet-portfolio](https://github.com/pharmacist-sabot/suradet-portfolio) - Professional portfolio showcasing projects and experience.
